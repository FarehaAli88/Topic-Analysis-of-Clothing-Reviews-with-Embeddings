# Topic Analysis of Clothing Reviews with Embeddings

### Project Summary
This repository contains a technical analysis of the "Women's Clothing E-Commerce Reviews" dataset. The project demonstrates an end-to-end pipeline for processing unstructured text data, utilizing OpenAI's embedding API to generate vector representations, and applying semantic similarity techniques to categorize customer feedback into actionable themes.

### Technical Skills
* **Natural Language Processing:** Text preprocessing, vectorization, and embedding generation.
* **AI Orchestration:** Integrating OpenAI APIs to perform semantic search and clustering.
* **Data Science Workflow:** Exploratory Data Analysis (EDA) and cleaning high-volume datasets.
* **Python Ecosystem:** Utilizing libraries for data manipulation and vector-based analysis.

### Approach
1. **Data Preprocessing:** Cleaning and normalizing raw review text to ensure high-quality input for the embedding models.
2. **Embedding & Analysis:** Utilizing pre-trained embedding models to convert text into multi-dimensional vectors.
3. **Clustering:** Implementing similarity search algorithms to group reviews based on shared thematic context rather than surface-level keyword frequency.

### Key Learnings
* Developed a deeper understanding of how high-dimensional vector spaces can be used to capture the semantic "meaning" behind customer feedback.
* Learned to optimize API calls for batch processing large datasets efficiently.
* Demonstrated the ability to transform unstructured sentiment data into clear, organized insights for business stakeholders.
